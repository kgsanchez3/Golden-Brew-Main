# The Golden Brew — Coffee Shop Website

## Project Overview
*The Golden Brew* is a cozy and aesthetic coffee shop website created as a school project by Katherine Sanchez, WD 304.  
The website showcases the café’s products, story, and contact information, using a blue and yellow color palette for a warm and inviting feel.  

This project demonstrates:
- HTML5 semantic structure  
- CSS design principles for color, layout, and typography  
- On-page SEO best practices (meta descriptions, long-tail keywords, alt attributes, title attributes, anchor text)  

---

## Pages Included
1. **Home** — Welcome message, featured coffee, and links to other pages.  
2. **Products** — Menu of drinks and items available.  
3. **About** — Background and vision of the café.  
4. **Contact** — Contact form and café information.  
5. **Our Story** — Section linked via anchor text “Our Story” describing the café’s journey.  

---

## Design Details
- **Color Palette:** Blue (#add8e6) & Yellow (#ffd700)  
- **Font:** Poppins, sans-serif  
- **Layout:** Clean, organized sections using `<header>`, `<main>`, `<section>`, `<footer>`  
- **Images:** All images include descriptive `alt` attributes  
- **Links:** Anchor tags include `title` attributes for accessibility  

---

## SEO Implementation
- **Meta Description:** Added for each page  
- **Long-tail keywords:** Specified in HTML comments before meta descriptions  
- **Anchor text:** “Our Story” links to a dedicated page  
- **Alt text for images:** Added to improve accessibility and SEO  

---

## How to Run the Website
1. Download or clone the repository to your local machine.  
2. Open the folder and locate the HTML files (`index.html`, `products.html`, etc.).  
3. Open the HTML file in a web browser (Chrome, Edge, or Firefox).  
4. The `style.css` file is linked externally and will automatically apply styles.
